# Sentinel-pme-Africa
Sentinel PME Africa est une plateforme IDS inspirée de Wazuh pour PME camerounaises. Agent Windows surveille réseau et détecte Nmap, Hydra et ransomware. Alertes envoyées à un SOC Ubuntu avec dashboard web en temps réel. Intègre détection phishing NLP (MTN/Orange), chiffrement RSA/AES et abonnement Mobile Money.
